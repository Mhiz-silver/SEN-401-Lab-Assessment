# Student Score Project

## Description

This project is a simple Python program that stores student names and scores. It displays the students and finds the highest and lowest scores using helper functions.

## Files

- **app.py** – Runs the program.
- **students.py** – Stores the student data.
- **utils/helpers.py** – Contains functions for finding the highest and lowest scores.
- **requirements.txt** – No external packages are required.

## How to Run

Open a terminal in the project folder and run:

```bash
python app.py
```
